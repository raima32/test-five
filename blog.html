Question-2: What is the difference between innerHTML,innerText,and textContent?

Answer: innerHTML,innerText,and textContent have important differences in how they interact with HTML

  • innerHTML when you need to set or get HTML content
  • innerText when you need the text exactly as it appears visually to the user
  • textContent for better performance when working with text-only content

    • Question-3: What is event delegation in the DOM?

    Answer: Event delegation is a technique instead of attaching event listeners to individual elements,you attach a single event listener to a parent element that watches for events bubbling up from its child element.

    Question-4: What is event bubbing in the DOM?

    Answer: Event bubbling is a key concept in the DOM event propagation model. Here's how it works: When an event occurs on an element (like a click on a button), the event doesn't just trigger on that specific element. Instead, the event "bubbles up" through the DOM hierarchy, triggering the same event on each parent element in sequence.
